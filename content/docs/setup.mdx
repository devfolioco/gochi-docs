Pick your OS and run one command. The installer pulls `make`,
`arduino-cli`, `bun`, clones the gochi repo into `./gochi`, and installs
the ESP32 core. After that you flash a tiny LED-blink sketch to confirm
the toolchain works.

<Callout type="info">
  This page gets your toolchain ready. The
  [Wiring](/docs/wiring) page is where you build the pet itself and flash
  the real firmware.
</Callout>

## 1. Run the installer

<Tabs items={['macOS / Linux', 'Windows']}>

<Tab value="macOS / Linux">

```sh
curl -fsSL https://gochi.in/install.sh | bash
```

</Tab>

<Tab value="Windows">

Open **PowerShell as Administrator** (Chocolatey needs it), then run:

```ps
powershell -c "irm gochi.in/install.ps1 | iex"
```

</Tab>

</Tabs>

The script is idempotent — anything you already have installed is left
alone. When it finishes you'll be inside `./gochi` with the full
toolchain in place.

<Callout type="info">
  Want to read the script first? It's the right thing to do.
  [`install.sh`](/install.sh) · [`install.ps1`](/install.ps1).
</Callout>

## 2. Plug in the ESP32-C3

A small **red power LED** should light up. Leave the board plugged in.

Use a **USB-C data cable**, not a power-only one. If the board doesn't
enumerate (`make ports` shows nothing), the cable is the first thing to
swap.

## 3. Verify flashing works

From inside the `gochi` repo:

```sh
make test-led
```

The port is auto-detected. If the upload hangs at *"Connecting…"*, hold
**BOOT**, tap **RESET**, release **BOOT**, and run it again. Once it
completes, the **blue LED** on the ESP32-C3 starts blinking — flashing
works.
